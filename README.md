## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Cloudflare account (for deployment)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/portafolio-jeremylive.git
   cd portafolio-jeremylive
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally (optional)**
   ```bash
   # For local development, you can serve the static files
   # or use a simple HTTP server
   npx http-server . -p 3000
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Deployment

The project is configured for deployment on Cloudflare Workers/Pages.

1. **Install Wrangler CLI**
   ```bash
   npm install -g wrangler
   ```

2. **Login to Cloudflare**
   ```bash
   wrangler auth login
   ```

3. **Deploy**
   ```bash
   npm run deploy
   ```

The deployment script will automatically build and deploy to Cloudflare Workers.
